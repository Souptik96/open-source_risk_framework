

5. **Test** your changes with:

   ```bash
   pytest tests/
   ```
6. **Commit and push** your changes

   ```bash
   git add .
   git commit -m "Add meaningful commit message"
   git push origin my-feature
   ```
7. **Submit a Pull Request** via GitHub
