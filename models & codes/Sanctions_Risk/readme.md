# Sanctions Risk Module

This module provides tools and templates for sanctions risk assessment, including:

- **Data ingestion scripts:** Load data from CSV or major cloud providers (AWS, GCP, Azure).
- **Model snippet:** Rule-based scoring model for prioritizing sanctions risk by country, watchlist match, and transaction type.
- **Usage example:** Generate a basic sanctions risk report.
- **MCP template:** Document the model context, assumptions, and methodology for audit and compliance.

**How to use:**
1. Ingest your data using one of the ingestion scripts.
2. Run the model to calculate risk scores.
3. Generate a report of the highest risk cases.
4. Document your model with the provided MCP template.

For more details, see the main project documentation.
